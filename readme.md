# HLF Production Demo

A production demonstration of Hyperleger Fabric using Terraform/Azure. 

_This demo is not affiliated with the used brands/companies._

## Commands

**Getting started**
`terraform init`

**Deploy infra**  
`terraform apply -auto-approve`  

**Destroy infra**  
`terraform destroy -auto-approve`
